## üì• Reading Input

### 1Ô∏è‚É£ Read `n` lines, each with two space-separated numbers

```cpp
int n;
cin >> n;
for (int i = 0; i < n; ++i) {
    int a, b;
    cin >> a >> b;
    // process a, b
}
```

### 2Ô∏è‚É£ Same as above, but store in an array of pairs (tuples of size 2)

```cpp
int n;
cin >> n;
vector<pair<int, int>> arr(n);
for (int i = 0; i < n; ++i) {
    cin >> arr[i].first >> arr[i].second;
}
```

### 3Ô∏è‚É£ Reading into a vector of tuples (for more than 2 values)

```cpp
int n;
cin >> n;
vector<tuple<int, int, int>> arr(n);
for (int i = 0; i < n; ++i) {
    int a, b, c;
    cin >> a >> b >> c;
    arr[i] = {a, b, c};
}
```

### 4Ô∏è‚É£ Read an entire line of integers

```cpp
string line;
getline(cin, line);        // may need an extra getline after cin >> n to flush newline
stringstream ss(line);
vector<int> nums;
int x;
while (ss >> x) nums.push_back(x);
```

### 5Ô∏è‚É£ Read a whole vector of `n` integers

```cpp
int n;
cin >> n;
vector<int> arr(n);
for (int i = 0; i < n; ++i) cin >> arr[i];
```

### 6Ô∏è‚É£ Read a matrix (n √ó m)

```cpp
int n, m;
cin >> n >> m;
vector<vector<int>> mat(n, vector<int>(m));
for (int i = 0; i < n; ++i)
    for (int j = 0; j < m; ++j)
        cin >> mat[i][j];
```

## üì§ Printing Output

### 1Ô∏è‚É£ Print array, each element on a new line

```cpp
for (int x : arr)
    cout << x << '\n';
```

### 2Ô∏è‚É£ Print array, space-separated on one line

```cpp
for (int i = 0; i < arr.size(); ++i) {
    if (i) cout << ' ';
    cout << arr[i];
}
cout << '\n';
```

### 3Ô∏è‚É£ Print array of pairs (each on its own line)

```cpp
for (auto [a, b] : arr)
    cout << a << ' ' << b << '\n';
```

### 4Ô∏è‚É£ Print array of tuples (using `get<>`)

```cpp
for (auto &[a, b, c] : arr)
    cout << a << ' ' << b << ' ' << c << '\n';
```

### 5Ô∏è‚É£ Print matrix

```cpp
for (auto &row : mat) {
    for (int j = 0; j < row.size(); ++j) {
        if (j) cout << ' ';
        cout << row[j];
    }
    cout << '\n';
}
```

## ‚öôÔ∏è Other Handy Patterns

### Read until EOF

```cpp

while (cin >> a >> b) {
    // process
}
```

### Read strings / characters

```cpp
string s;
cin >> s;  // reads one word
getline(cin, s); // reads a full line (can include spaces)
```

### Print with formatting

```cpp
cout << fixed << setprecision(6) << 3.1415926535 << '\n';
```
